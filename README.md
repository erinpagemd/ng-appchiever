# Full-Contact

## Includes the following technologies:

### Config
* Gulp
* Bower for front-end dependencies
* Node/NPM for additional dependencies

### Style
* Sass
* Bootstrap
* Font-Awesome

### MV*
* Jade
* Angular
* Angular-UI-Router

### NoSQL Database
* Firebase



## To Use:

###Use the following commands:
```
npm install
bower install
firebase init
gulp
```
In separate terminal window use

```
python -m SimpleHTTPServer
```
